rule

Règles

Le but du jeu est de trouver le mot caché en 6 essais maximum ! Le mot caché est un mot français présent dans notre dictionnaire.

Pour trouver ce mot, vous devez simplement entrer un mot existant dans notre dictionnaire. Si le mot est valide, des indications de couleurs apparaitront :

  • Bravo, vous avez trouvé une bonne lettre bien placée
  • Cette lettre existe mais est mal placée
  • Cette lettre n'existe pas dans le mot recherché
